# Solid App Store

An app store for Solid that displays developer tools and apps that exist in the Solid ecosystem

## Setup and run HTTP server

Install dependencies:
```shell script
$ npm install
```

Run development server:
```shell script
$ npm run watch
```

Run production sever:
```shell script
$ npm start
```
